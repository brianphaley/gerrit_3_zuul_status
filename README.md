# gerrit_3_zuul_status
Dynamic Zuul results table in Gerrit 3

This Greasemonkey/Tampermonkey script shows Zuul results and live progress on
a gerrit review. It is geared towards OpenStack and OpenDev developers.

This was updated by several people ([1],[2],[3]) and this repository has
additional changes to modify the highlight colors slightly.
